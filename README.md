💸 FundFlow
Smart, lightning-fast, and zero-friction cash flow management.

FundFlow is a Full-Stack personal finance web application designed to break the tedium of manual data entry. Instead of complex forms and endless clicks, FundFlow leverages Smart UX logic—such as Smart Defaults, Quick Adds, and keyword-based auto-categorization—to make logging expenses and income an almost instant process.

The app was built with a mobile-first philosophy, ensuring a flawless experience whether you are logging on the go from your phone or tracking from your desktop.

✨ Key Features
Zero-Friction Manual Entry: Frontend intelligence that detects keywords in the description (e.g., "Uber") and automatically assigns the correct category.

Smart Defaults: Pre-configured fields (like today's date and the "Expense" type) to save users from unnecessary clicks.

Analytical Dashboard: Dynamic charts (Donut and Area charts) for tracking expenses by category, overall balances, and subscription alerts.

100% Responsive Interface: Clean and adaptable design for any screen size (Mobile, Tablet, and Desktop).

🛠️ Tech Stack & Deployment
FundFlow's ecosystem is distributed in a modern and scalable way:

Frontend (React + Tailwind CSS): Hosted on Vercel for instant, global loading times.

Backend (Python + FastAPI): High-performance asynchronous API hosted on Render.

Database (PostgreSQL): Robust relational database provided by Supabase.

🚀 How the Smart UX Works
Most finance apps fail because they demand too much work from the user. FundFlow solves this in the code:

The user opens the app on their phone (the date is already set to today, and the type is already set to "Expense").

They type "Lunch" and enter the amount.

The system recognizes the text, automatically switches the category to "Food", and it's done. A process that usually takes 15 seconds now takes 3.
